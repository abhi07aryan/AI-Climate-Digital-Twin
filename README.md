# AI-Powered Digital Twin of India's Climate

## Objective

Develop an AI-powered Digital Twin capable of

- Climate State Reconstruction

- Rainfall Forecasting

- Temperature Forecasting

- Scenario Simulation

using IMD and INSAT datasets.
